# Telegram Leads Bot

Telegram-бот для автоматического приёма заявок и сохранения их в Google Sheets.

## Возможности
- Сбор имени, телефона и комментария
- Автоматическая запись в Google Sheets
- Подходит для малого бизнеса

## Стек
- Python
- pyTelegramBotAPI
- Google Sheets API

## Как запустить
1. Создать Telegram-бота через BotFather
2. Получить credentials.json (Google Cloud)
3. Включить Google Sheets API и Google Drive API
4. Установить зависимости
5. Запустить bot.py
